# Memory Card Game

**[Live Preview](https://memory-card-game-eight-livid.vercel.app/)**

This is a Memory Card game built using React as part of [The Odin Project's curriculum](https://www.theodinproject.com/lessons/node-path-react-new-memory-card). The goal of the game is to click on as many unique cards as possible without clicking the same card twice. Every time a card is clicked, the positions of the cards are randomized.
